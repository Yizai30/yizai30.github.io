### 常用组件
1. uvx
- 主要用途：直接运行 Python 工具/脚本：无需先通过 pip 安装包，可直接从网络（如 PyPI）获取并执行代码。【by Deepseek】
- 使用场景：通过 uvx 配置 Claude Code 的 mcp 工具，则无需在本地安装，可以直接通过 PyPI 运行 server。
- 示例：
```json
{
  "mcpServers": {
    "ppt": {
      "command": "uvx",
      "args": [
        "--from", "office-powerpoint-mcp-server", "ppt_mcp_server"
      ],
      "env": {}
    }
  }
}
```配置文件的位置，若系统是 macOS/Linux，则位于 ~/.claude.json，若系统是 Windows，则位于 %USERPROFILE%\.claude.json。
