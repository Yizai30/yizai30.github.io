### 常用组件
#### 1. uvx
- 主要用途：直接运行 Python 工具/脚本：无需先通过 pip 安装包，可直接从网络（如 PyPI）获取并执行代码【by Deepseek】。
- 使用场景：通过 uvx 配置 Claude Code 的 mcp 工具，则无需在本地安装，可以直接通过 PyPI 运行 server。示例：

    ```json
    {
        "mcpServers": {
            "ppt": {
            "command": "uvx",
            "args": [
                "--from", "office-powerpoint-mcp-server", "ppt_mcp_server"
            ],
            "env": {}
            }
        }
    }
    ```
    上面的配置文件的位置，若系统是 macOS/Linux，则位于 [`~/.claude.json`](https://zhuanlan.zhihu.com/p/1933624323849032925)，若系统是 Windows，则位于 `%USERPROFILE%\.claude.json`。

#### 2. vcpkg
- 主要用途：[管理 C/C++ 库](https://learn.microsoft.com/zh-cn/vcpkg/get_started/overview)，包括：
    1. 包管理（提供超过 2300 个[开源库](https://github.com/microsoft/vcpkg/tree/master/ports)，包括 cpprestsdk、eigen3 等，跨平台支持 Windows/macOS/Linux）。
    2. 版本控制（防止版本冲突、一个系统上同一个库的多个版本共存）。
    3. 二进制缓存（避免重复生成以提高构建速度）。
- 使用场景：
    vcpkg 让 C++ 依赖管理变得简单，就像 Node.js 的 npm 或 Python 的 pip 一样【by Claude Code】。在项目根目录下创建 `vcpkg.json`，执行 `vcpkg install --triplet=x64-windows` 安装库，之后可以使用 cmake 配置文件 `CMakeLists.txt` 引用库。示例：

    vcpkg.json 如下：
    ```json
    {
        "name": "planegcs",
        "version": "1.0.0",
        "description": "...",
        "homepage": "...,
        "dependencies": [
            {
                "name": "eigen3",
                "version>=": "3.4.0"
            },
            {
                ...
            }
        ],
        "builtin-baseline": "5dfa11cf88a8da8877b54fcbc5f6db78bc13f75a"
    }
    ```
    上面的 “builtin-baseline” 可以通过 `vcpkg x-update-baseline` 命令刷新。
    
    CMakeLists.txt 则可以通过 find_package 找到库。如下：
    ```cmake
    ...
    find_package(Eigen3 REQUIRED)

    ...
    message("Eigen3 version: ${Eigen3_VERSION}")
    message("Eigen3 include dirs: ${Eigen3_INCLUDE_DIRS}")

    ...
    add_executable(llm_geometry_animation)

    ...
    target_link_libraries(llm_geometry_animation
        Eigen3::Eigen
    )
    ```
